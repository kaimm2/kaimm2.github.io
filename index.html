Why Create an Account?
  • Permanent Storage: Guest files expire in 24h. User files last forever.
  • Private Mode: Hide your scripts from the public list.
  • API Access: Upload/Obfuscate directly from Roblox/Termux.
  • No Limits: Unlimited obfuscation (Guests limited to 5/day).

How to Use

  • Upload & Folders:
    Enter a filename like MyScript.lua. To create a folder, use slashes: Hubs/V1/Main.lua.
  • Obfuscation:
    Paste your code, select a preset (e.g., 'Evil' or 'Minify'), and click "Obf".
    Now powered by Fast Node.js API.
  • API & Automation:
    Click the icon (Settings) after logging in to get your API Token and code examples for Roblox executors.
  • Raw Access:
    Use the switch in the file list to temporarily disable raw access (useful for maintenance).
